# 英単語学習ドリルアプリ

大学受験対応の英単語学習ドリルアプリです。

## 🚀 デモ

[https://shotadaikohara.github.io/english_word_test/](https://shotadaikohara.github.io/english_word_test/)

## ✨ 主要機能

### 📚 学習機能
- **4択問題形式**での英単語学習
- **3段階の難易度**（基礎・標準・上級）
- **5秒タイマー**で回答速度を向上
- **苦手問題の自動抽出**と再出題

### 📊 分析機能
- **成績分析**: 学習成績の推移をグラフで表示
- **速度分析**: レベル別・セッション別の回答時間分析
- **単語別分析**: 各単語の詳細データをグラフで可視化
  - 散布図（正答率 vs 回答時間）
  - 棒グラフ（正答率順）
  - バブルチャート（出題回数込み）

### 🎯 大学受験対応
- **志望大学設定**（東大・京大・早稲田・慶應・上智）
- **合格可能性計算**（正答率と回答速度から算出）
- **大学別合格基準**に基づく学習指導

### 🔍 高度な機能
- **単語検索・フィルタリング**
- **学習履歴の自動保存**（ローカルストレージ）
- **レスポンシブデザイン**（スマホ対応）
- **視覚的なフィードバック**（色分け・アニメーション）

## 🛠️ 技術仕様

- **フロントエンド**: HTML5, CSS3, JavaScript (ES6+)
- **グラフライブラリ**: Chart.js
- **データ保存**: LocalStorage
- **デザイン**: レスポンシブデザイン

## 📱 使い方

1. **設定**: 志望大学と難易度レベルを選択
2. **ドリル**: 5秒以内に4択問題に回答
3. **分析**: 学習データをグラフで確認
4. **復習**: 苦手問題を重点的に学習

## 🎨 特徴

- **直感的なUI**: わかりやすいナビゲーションとデザイン
- **リアルタイムフィードバック**: 即座に正解・不正解を表示
- **学習継続をサポート**: 進捗管理と目標設定機能
- **データ駆動**: 詳細な分析で効率的な学習をサポート
